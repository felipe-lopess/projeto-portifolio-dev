<h1>Visão Geral</h1>

Este projeto pratico do curso Fullstack Rocketseat, é um site do  de portfólio pessoal construído com HTML e CSS puros. Ele apresenta informações sobre o desenvolvedor, projetos em destaque, serviços oferecidos e contatos, com foco em uma interface moderna e visualmente consistente.

<h2>Estrutura do projeto</h2>
index.html --- Página principal do portfólio: Contém todas as seções do site: perfil, projetos, serviços e contato.
<br>
index.css --- Arquivo CSS principal que importa estilos globais e de seções.
<br>
global.css --- Define variáveis de design, cores, tipografia e reset básico.
<br>
sections.css --- Estilos específicos para cada seção do site
Inclui layout de cards, seção de destaque, serviços e contato.
